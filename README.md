二、详细过程说明（专业学习—《机器学习》学习笔记）

本次重点阅读周志华《机器学习》，系统搭建机器学习知识框架，笔记如下：
本书全面覆盖机器学习核心体系，从基础概念到经典算法，再到进阶理论，逻辑严谨、实例通俗，是数据分析师必备的理论基石。书中明确机器学习本质是用数据驱动模型、用经验优化性能，通过学习算法从数据中挖掘规律，实现预测、分类与决策。
我重点学习了监督学习、无监督学习、模型评估三大模块。监督学习部分掌握线性回归、逻辑回归、决策树、支持向量机等核心算法，理解线性模型的拟合原理、决策树的分裂与剪枝策略、SVM的核技巧与软间隔优化，明白算法如何处理分类与回归任务。无监督学习聚焦K-Means聚类、PCA降维，掌握无标签数据的模式挖掘与高维数据简化方法，对数据预处理、特征工程极具指导意义。
模型评估与选择章节让我建立科学的模型评价思维，掌握训练集/验证集/测试集划分、交叉验证、精确率/召回率/F1值、ROC/AUC等指标，理解过拟合与欠拟合成因及正则化、剪枝等优化手段。此外，集成学习、贝叶斯分类等内容，帮助我理解多模型融合提升效果的思路，为后续业务建模、特征选择提供理论支撑。
全书既夯实数学原理，又贴近工程实践，让我能从理论层面解释数据分析中的建模逻辑，显著提升算法理解与模型优化能力。
